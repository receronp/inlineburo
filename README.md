# inlineburo.com — redesign

Static site built with Astro + Tailwind. Design direction, audit, and content
grounding live in [REDESIGN.md](REDESIGN.md) and `docs/content/`.

## Develop

```sh
npm install
npm run dev      # local dev server
npm run build    # static build to dist/
npm run preview  # serve the build
```

## Content rules

All copy derives from `docs/content/post-inventory.md`. Client names marked
`[por confirmar]` in `docs/content/project-stories.md` must be confirmed before
publishing. No invented metrics or dates.
